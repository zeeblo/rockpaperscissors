# rockpaperscissors
A rock paper scissors static website: https://zeeblo.github.io/rockpaperscissors/

# Overview

It's pretty straight forward if you know the rules of "rock paper scissors" or "Jankenpon" for any Japanese speakers out there.

Basic Rules:

**Rock > Scissors**  <br>
**Scissors > Paper**  <br>
**Paper > Rock**
